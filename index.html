IntelliJ Platform SDK Documentation

Welcome to the IntelliJ Platform SDK Documentation!

Here you can find basic information about the IntelliJ Platform SDK, in particular how to get started with developing plugins for IntelliJ IDEA and other JetBrains IDEs.

More details about specific development topics can be found in the Architecture Reference Guide.

A set of step-by-step tutorials with code samples will lead you through the common use cases for the IntelliJ Platform SDK.

The Plugin Development FAQ section contains a series of issues commonly discussed on our Open API and Plugin Development forums.
